# BeautyShop Demo - Vanguardia Estética

This is a professional and modern demo website for an aesthetics clinic, built with React, TypeScript, and Vite.

## 🚀 Live Demo
[View Live Demo](https://JulianLechuga.github.io/BeautyShop-demo/)

## ✨ Features
- **Premium Design**: Gold and beige color palette for a luxury aesthetic.
- **Service Grid**: Highlighting treatments like Radiofrecuencia, Presoterapia, and Cavitación.
- **Responsive**: Fully optimized for mobile and desktop devices.
- **WhatsApp Integration**: Floating button for direct contact.

## 🛠️ Tech Stack
- React 19
- TypeScript
- Vite
- Lucide React (Icons)
- CSS Modules / Vanilla CSS

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/JulianLechuga/BeautyShop-demo.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
